# 🎬 Streamflix

A fully modern OTT streaming platform UI — built with **React** and inspired by platforms like **Netflix**, **Hotstar**, and **Amazon Prime Video**.  
But with my own creative twist — **neon pastel aesthetic**, smooth animations, and unique components.

---

## 🧠 Features

- 🎞️ **Auto-rotating Hero Banner** – changes every 60 seconds with dynamic movie previews
- 🎬 **Movie Cards** – clean hover animations, click to view modal
- 🔍 **Live Search with Preview** – shows results as you type
- 🎯 **Genre & Type Filters** – filter by movie, TV shows, web series, etc.
- 🎨 **Custom Pastel + Neon UI Theme**
- 📱 **Fully Responsive** – mobile-friendly layout & filters
- 🧩 **7+ Components** – modular, scalable React architecture
- 🌟 **Font Awesome Integration**
- ✨ **Custom Scrollbar Styling**
